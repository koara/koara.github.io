What is Koara?

Many applications use some sort of lightweight markup language to enhance the writing experience of their users. But not every use case requires the same type of syntax. This has led to many forks and flavors of existing languages.

Koara is a modular lightweight markup language. The application owner can decide which parts of the syntax they want to allow. When a user writes syntax that is not allowed, it will just render as plain text.
